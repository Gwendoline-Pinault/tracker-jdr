# Tracker JDR

Ce projet personnel consiste à suivre mes campagnes de jeux de rôle et d'en établir les statistiques.

## Historique

Ce projet a été construit progressivement. Initialement un simple script Python lancé depuis mon terminal, je l'ai amélioré afin de créer un site web permettant aux joueurs de consulter leurs statistiques. J'ai ajouté également suite à leurs retours, une page détail pour chaque personnage avec des statistiques plus complètes.

Ce dépôt actuel est la deuxième version du site, la première étant codée uniquement avec Python pour le programme et du JavaScript natif pour l'affichage de la page web. Je lançais initialement le script à la main après chaque partie pour calculer les statistiques des dés obtenus. Avec la version actuelle, je peux saisir directement la partie depuis le site.

## Stack
Le serveur a été fait avec EJS - express - Node.js. J'utilise également Python pour générer les histogrammes.

## Déploiement
Le site est déployé avec Vercel.
